# 🇸🇬 賴皮家族新加坡之旅 | Singapore Trip

一個為家族旅行設計的響應式 Web 應用程式，提供完整的行程規劃、備案方案、伴手禮建議與實用工具。

**線上體驗：** https://lai-wen.github.io/singapore-trip/

---

## ✨ 功能特色

### 📅 五大主要功能

1. **行程總覽** - 8 天完整行程一目了然
2. **每日行程** - 時間軸式詳細規劃，含地圖導航
3. **備案行程** - 雨天/熱天/體力調整替代方案
4. **伴手禮清單** - 精選新加坡特色商品推薦
5. **重要必讀** - 匯率轉換、辦事處資訊、行李寄放

### 🎯 設計亮點

- **長輩友善** - 大按鈕（最小 44px）、高對比度、清晰排版
- **地圖整合** - 一鍵導航至 Google Maps
- **匯率工具** - 即時換算新幣 ↔ 台幣（固定匯率 1:24）
- **響應式設計** - 完美支援手機、平板、桌面

## 🚀 快速開始

### 本地運行

```bash
# 1. 安裝依賴
npm install

# 2. 啟動開發伺服器
npm run dev

# 3. 開啟瀏覽器
http://localhost:3000
```

### 生產構建

```bash
# 編譯生產版本
npm run build

# 預覽生產版本
npm run preview
```

## 📂 專案結構

```
singapore-trip/
├── App.tsx              # 主要元件（所有 Tab）

├── types.ts            # TypeScript 類型定義

├── vite.config.ts      # Vite 設定

    └── workflows/
```
